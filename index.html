Tillgänglighet

Vad betyder tillgänglighet ?

Tillgänglighet metoden betyder att göra webbplatser användbara för så många människor som möjligt. Traditionellt sätt tillgänglighet handlar om personer med funktionsnedsättning, men metoden att göra webbplatser tillgängliga till andra grupper av människor också, de som använder mobila enheter eller de med långsamma nätverksanslutningar.

Vad tillgänglighet på webben innebär?

Tillgängligt på webben ger stora möjligheter att tillhandahålla information och tjänster utan att diskriminera användare.Vi kan förklara det mer specifikt t.ex. uppfatta, förstå, navigera och interagera med webben bidra till webben. Webbtillgänglighet omfattar alla funktionshinder som påverkar tillgången till webben, inklusive: auditiv, kognitiv,neurologiska, fysisk och Tal, visuell.
Webbtillgänglighet gynnar också människor utan funktionshinder, till exempel: personer som använder mobiltelefoner, smarta klockor, smarta TV-apparater och andra enheter med små skärmar, olika inmatningslägen osv.

Fem tillgängliga konkreta exempel design på en webbplats

  1. Lägg till tillräckligt med färgkontrast så att personer med nedsatt syn kan använda
  2. Använd inte färg enbart för att göra viktig information begriplig
  3. Se till att interaktiva element är lätta att identifiera
  4. Ge tydliga och konsekventa navigeringsalternativ
  5. Ensure that form elements include clearly associated labels

Fem konkreta exempel på tillgänglig kod.

  1. Skriva kod som anpassar sig till användarens teknik
  2. Se till att alla interaktiva element är tillgängliga med tangentbordet
  3. Identifiera sidspråk och språkändringar
  4. Help users avoid and correct mistakes
  5. Se till att ordningen på elementen i koden matchar den logiska ordningen för informationen som presenteras
